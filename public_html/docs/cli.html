jspm cli

Create import maps for arbitrary JavaScript modules, and run them in any environment.

The JSPM CLI is a command-line tool for managing the dependencies of your JavaScript modules using import maps, allowing them to be run in environments such as the browser or Deno with minimum fuss. JSPM is standards-compliant and designed to be compatible with npm, supporting features like:

  • Local Linking: resolve against node_modules for local development workflows
  • Common CDNs: plugins for a wide range of CDNs, such as jspm.io, UNPKG, Skypack and more
  • Conditional Resolution: environment-specific module linking
  • Dependency Versioning: respects package.json dependency ranges
  • Package Entrypoints: node-style package exports, imports and own-name resolution

Getting Started

To learn how you can start integrating the JSPM CLI into your workflow, check out the Getting Started guide.
