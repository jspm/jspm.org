JSPM provides a module CDN and package management for import maps, allowing
any package from npm to be loaded directly in the browser fully optimized without further tooling.

March 2nd 2021: The new ga.jspm.io import map CDN has been released, alongside the landing of native import maps in Chrome 89+. See the release post for details.
